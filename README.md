# Guangyuan Xiaolongbao Proposal

Vertical customer proposal for a private Shanghai food culture experience:
Fuchun Xiaolong, Guangyuan Wet Market, and a Shanghainese home-style kitchen.

## Files

- `guangyuan-xiaolongbao-proposal.html` - mobile-first HTML proposal page.
- `assets/guangyuan-xiaolongbao/` - image assets used by the proposal.
- `exports/guangyuan-xiaolongbao-vertical-proposal.pptx` - editable vertical PPT version.
- `tools/export_guangyuan_vertical_ppt.py` - script for regenerating the PPTX from the local assets.

## Notes

The generated `outputs/` working directory is intentionally ignored.
